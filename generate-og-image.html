🖼️ og-image.png Generator

预览和下载用于社交媒体分享的分享图（1200×630）。点击 Download PNG 保存到本地，然后上传到网站根目录。

✓ 生成后把 og-image.png 放到 GitHub 仓库根目录（和 index.html 同级）。部署后 https://theouyi.com/og-image.png 就会被社交媒体抓取。
💡 一些替代方案：
• 用 cloudconvert.com 把 og-image.svg 直接转 PNG
• 用 Figma / Sketch 打开 SVG 导出 PNG
• SVG 本身也可以作为 og:image，现代浏览器都支持，但 Twitter 卡片兼容性不如 PNG
